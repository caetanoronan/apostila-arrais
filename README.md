# Apostila Arrais

Este repositório contém a versão clonada e atualizada da apostila (`Apostila_arras_teste_08_clone.html`) com imagens e recursos necessários para publicação via GitHub Pages.

Como usar
- Abra `Apostila_arras_teste_08_clone.html` localmente para revisão.
- Para publicar no GitHub Pages, adicione um repositório remoto e faça push das branches `main` e `gh-pages`.

Sugestões
- Se for publicar com GitHub Pages, considere usar a branch `gh-pages` ou ativar Pages a partir da `main`/`docs`.
- O arquivo `.nojekyll` está incluído para evitar processamento Jekyll em nomes com underscores.

Contato
- Atualize este README com seu nome/contato e instruções adicionais conforme necessário.
